## Active Task Contract
- `docs/tasks/` keeps only active or incomplete task docs with status `todo`, `doing`, or `blocked`.
- When a task becomes `done`, summarize its lasting outcome in `../tracking/completed-work-summary.md` and delete the completed task doc.
- Completed task history is intentionally pruned so future agents do not accumulate an unbounded default reading set.
